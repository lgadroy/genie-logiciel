# Projet GÉNIE-LOGICIEL

## Interface web pour la visualisation de jeux de données

### Description du projet

Réalisation d'une carte interactive de la France. Nous pourrons voir les gares des départements français et faire des classements des plus propres au plus sales. Nous nous baserons sur des jeu de données qui sont donnée par la SNCF et ces jeux de données sont en libre accès.

### Installation de l'application

#### Clone du dépôt

```bash
git clone https://github.com/lgadroy/genie-logiciel.git
cd genie-logiciel
```

### Commande GIT :

```bash
git add .                           # Ajoute tous les fichiers/dossiers créés ou modifiés à l'index

git push                            # Envoie les commits vers le dépôt distant (GitHub)

```
